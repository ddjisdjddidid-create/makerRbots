# Telegram Bot Factory - مصنع بوتات تليجرام

## Overview
مصنع بوتات تليجرام لإنشاء ثلاثة أنواع من البوتات:
1. **بوت ذكاء اصطناعي** - متصل بـ Groq API مع دعم تحليل الصور
2. **بوت أذكار** - ينشر الأذكار في المجموعات والقنوات
3. **بوت منع تصفية** - حماية المجموعات والقنوات من المخربين

## Project Structure

```
├── bot.py              # الكود الرئيسي للمصنع
├── render.yaml         # إعداد Render للنشر
├── Procfile            # للتشغيل على Render
├── requirements.txt    # المكتبات المطلوبة
├── member.json         # إحصائيات المستخدمين
├── remember.json       # ذاكرة الذكاء الاصطناعي
├── guard_data.json     # بيانات بوتات الحماية
├── guard_admins.json   # المشرفين المراقبين
├── kick_counts.json    # عداد الطرد لكل مشرف
└── replit.md           # توثيق المشروع
```

## Features

### المصنع الأساسي
- ترحيب بأزرار انلاين لاختيار نوع البوت
- اشتراك إجباري في @TepthonHelp
- لوحة تحكم للمطور @Dev_Mido (أمر: التحكم)
- دعم الانلاين
- إحصائيات البوتات المصنوعة

### بوت الذكاء الاصطناعي
- ربط بـ Groq API (llama-3.3-70b-versatile)
- دعم تحليل الصور (Llama 4 Scout Vision)
- ذاكرة للمحادثات (آخر 20 رسالة)
- لوحة إدارة للمالك (حظر/فك حظر/إذاعة)
- تغيير قناة الاشتراك الإجباري
- إحصائيات

### بوت الأذكار
- نشر أذكار للقنوات والمجموعات
- اختيار فترة النشر (5 دقائق، ربع ساعة، ساعة، الخ)
- اختيار مدة النشر (يوم، أسبوع، دائم)
- مجموعة كبيرة من الأذكار والآيات القرآنية (85+ ذكر)
- شكل نظيف بدون ايموجي

### بوت منع التصفية (جديد)
- حماية المجموعات والقنوات من المخربين
- تتبع عدد طرد الأعضاء لكل مشرف
- تنزيل المشرف تلقائياً عند تجاوز حد التصفية
- أوامر رفع/تنزيل المشرفين
- إعدادات قابلة للتخصيص (حد التصفية)
- لوحة تحكم للمالك مع إحصائيات

## Environment Variables

| Variable | Description |
|----------|-------------|
| BOT_TOKEN | توكن البوت الرئيسي من @BotFather |
| GROQ_API_KEY | مفتاح API من Groq |

## Deployment on Render

1. انشئ حساب على render.com
2. اختر Web Service
3. اربط المشروع من GitHub
4. أضف Environment Variables:
   - BOT_TOKEN: توكن البوت
   - GROQ_API_KEY: مفتاح Groq
5. اختر Free Plan
6. Deploy
